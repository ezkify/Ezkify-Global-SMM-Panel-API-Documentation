# Security Policy - Ezkify Global

## Supported Versions
We actively provide security updates for the current API v2.

| Version | Supported          |
| ------- | ------------------ |
| v2      | :white_check_mark: |
| v1      | :x:                |

## Reporting a Vulnerability
At Ezkify Global, we take the security of our 50,000+ trusted agencies seriously. If you discover a security vulnerability, please do not open a public issue. Instead, send an email to:

**ezkify@proton.me**

Please include:
* A description of the vulnerability.
* Steps to reproduce.
* Your contact information for follow-up.

We aim to respond to all security reports within 24-48 hours.
