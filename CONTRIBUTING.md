# Contributing to Ezkify Global API

Thank you for your interest in improving the Ezkify Global developer ecosystem!

## How to Contribute
1. **Report Bugs:** Open an issue with the "bug" label.
2. **Feature Requests:** Suggest new AI-safe growth services by opening an issue.
3. **Pull Requests:** * Fork the repo.
   * Create a branch for your feature.
   * Ensure your code follows PSR standards (for PHP) or PEP8 (for Python).
   * Submit the PR for review.
